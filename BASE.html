Algoritmos no dia-a-dia

Algoritmos

Algoritmo é uma sequência finita de instruções, que devem ser seguidas para resolver situações em um espaço de tempo e com esforço finito, de forma que se resolva a situação. A resolução do problema é feita mecanicamente ou eletrônicamente.
- Professor Gustavo Guanabara - Algoritmos

Algoritmos estão presentes em nosso dia a dia, dês de quando nosso cérebro nos manda instruções para resolver algum ''problema'' cotidiano, até quando nosso celular pede algum login, por exemplo.

Exemplo de uma sequência finita no dia-a-dia de um músico

TOCAR GUITARRA

  1. Pegar o instrumento

      2. Se o instrumento estiver afinado, prossegue ao passo 2, caso contrário, afina antes.

  2. Plugar no amplificador

      3. Se o amplificador estiver ligado na tomada, prossegue ao passo 3, caso contrário, liga na tomada antes.

  3. ligar distorção

      4. Se a distorção estiver regulada, prossegue ao passo 4, caso contrário, regula primeiro.

  4. Tocar

      5. Se encerra a sequência.

FIM
